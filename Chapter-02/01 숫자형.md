## 배운 것
* ;(콜론)이 필요없다
* 자료형 선언이 필요없다
* type(a) 자료형을 체크하는 방법
* '/' 나누기하면은 실수형으로 바로 나타낸다.
* C나 자바에서 보던것처럼 몫을 보고싶으면 '//' 두개가 필요하다.
* 곱하기를 두개하면 제곱이다

## 코드
```py
a=1
b=1.3
c= 4.24e10

e=4
f=5

print(type(a)) 
print(type(b))
print(c)

print(e/f)
print(e//f)
print(e%f)
print(e ** f)
```

## 출력
```
<class 'int'>
<class 'float'>
42400000000.0
0.8
0
4
1024
```
